# Subjective-well-being-in-Colombia

## Description
Analysis of subjective well-being determinants in Colombia using the National Quality of Life Survey (ENCV - Encuesta Nacional de Calidad de Vida) for the period 2018-2024.

## Project Structure
- `01_data/`: Raw and processed datasets
- `02_scripts/`: Analysis code and functions
- `03_outputs/`: Results, figures, and tables
- `04_docs/`: Documentation and manuscript

## Usage
1. Run scripts in numerical order
2. Results are automatically saved to 03_outputs/

## Data Source
National Quality of Life Survey (ENCV) - Colombian National Statistics Department (DANE), years 2018-2024.
